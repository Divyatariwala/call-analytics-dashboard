Call Analytics Dashboard
Project Description

The Call Analytics Dashboard is an interactive web application designed to visualize and analyze call data efficiently. It provides real-time insights into call activity, costs, and geographic distribution, helping businesses and teams monitor performance, identify trends, and make data-driven decisions. The dashboard is fully responsive and works seamlessly across devices, including desktops, tablets, and mobile phones.

Key functionalities include:

Total Calls Overview: Quickly see the total number of calls.

Calls by City: Visualize call distribution across different cities using a pie chart.

Activity Timeline: Track hourly or daily call trends using line charts.

Cost Analysis: Compare the total cost per city using bar charts.

Responsive Design: Optimized for mobile and desktop devices.

Technology Stack

Frontend: React.js

Styling & Layout: Tailwind CSS

Charts & Data Visualization: Recharts (Pie, Line, and Bar charts)

Deployment: Vercel

Version Control: Git & GitHub

Features

Interactive Charts: Hover effects and tooltips to explore data points.

Dynamic Data: Charts update automatically based on the provided dataset.

Responsive Design: Mobile-first approach ensures readability and usability on smaller devices.

Customizable UI: Colors, fonts, and chart labels can be easily adjusted.

Performance Metrics: Provides key metrics like total calls and total costs at a glance.
